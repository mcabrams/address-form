#### Developing locally

First time? Make sure you have node installed. Then:

```$ npm install --global gulp```

Then, to install dependencies, navigate to the root of this repo dir and run:

```$ npm install```

Then:

```$ bower install```

And finally:

```$ gulp serve```

You should be good to go.
